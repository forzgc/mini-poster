## 特性

- 使用 TypeScript 编写，提供完整的类型定义
- 使用 新版 canvas 2d 接口，性能更佳

## 安装

使用 `pnpm` 安装:

```bash
pnpm add @inottn/miniposter
```

使用 `yarn` 或 `npm` 安装:

```bash
# 使用 yarn
yarn add @inottn/miniposter

# 使用 npm
npm i @inottn/miniposter
```
